- ဒီနေ့ HTML အကြောင်း သင်ကြရမယ်။ HTML ဆိုတာ ဝက်ဘ်ဆိုဒ်တွေ ဆောက်တဲ့အခါမှာ အခြေခံအုတ်မြစ်လိုပါပဲ။ ဥပမာ အိမ်ဆောက်မယ်ဆိုရင် အုတ်နဲ့ဘိလပ်မြေလိုသလိုပေါ့။ - HTML က HyperText Markup Language ဖြစ်တယ်၊၊ Programming Language မဟုတ်ပါဘူး။ - HTML 1.0: 1991 (invention by Tim Berners-Lee) / 1993 (release) - HTML 2.0: 1995 - HTML 3.2: 1997 - HTML 4.0: 1997 (also referenced as 1999 in some sources) - HTML5: 2014
